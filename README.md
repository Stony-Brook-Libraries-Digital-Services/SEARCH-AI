# SEARCH AI  

**Advancing Academic Library Discovery with Artificial Intelligence**  

Stony Brook University Libraries is developing **SEARCH AI**, an AI-powered search tool designed to transform how users engage with the library catalog. This initiative enabling users to interact with a full library catalog using **natural language queries** instead of relying solely on traditional keyword searches.
