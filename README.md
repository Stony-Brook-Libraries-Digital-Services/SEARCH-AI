SEARCH AI

Advancing Academic Library Discovery with Artificial Intelligence

Stony Brook University Libraries is developing SEARCH AI, an AI-powered search tool designed to transform how users engage with the library catalog. This initiative represents the first of its kind in a research library environment, enabling users to interact with the catalog using natural language queries instead of relying solely on traditional keyword searches.
